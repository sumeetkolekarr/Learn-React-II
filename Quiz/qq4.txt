1. How would you describe the concept of 'State'?
    A way for React to remember saved values from within a component. This is similar to declaring variables from within a component.

2. When would you want to use props instead of state?
    Anytime you want to pass data into a component so that component can determine what will get displayed on the screen

3. When would you want to use state instead of props?
    Anytime you want a component to maintain some values from wothin a component and remember those values even if the React re-renders the component

4. What does "immutable" mean? Are props immutable? Is state immutable?
    Unchanging. Props are immutable. State is mutable
